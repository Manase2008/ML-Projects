# ML-Projects

Welcome! This repository is a collection of my projects where I am learning and experimenting with different Machine Learning algorithms using Python and Google Colab.

## 🧠 Algorithms Explored

### 🔹 Linear Regression
* **Project:** Predicting Californian House Prices
* **What I did:** Used a dataset of housing features to predict prices. Learned how to fit a trend line and evaluate it using Mean Squared Error (MSE), mean absolute error (MAE) and r2 score (r2).
* **Notebook:** [https://colab.research.google.com/drive/1pWC5tyU9RBrBzIafih644NXBf1h9GbPR]

### 🔹 Logistic Regression
* **Project:** Prediction of Loan Risk.
* **What I did:** Built a binary classifier to detect the risk of taking the loan.
* **Notebook:** [https://colab.research.google.com/drive/1pWC5tyU9RBrBzIafih644NXBf1h9GbPR]

### 🔹 Decision Trees
* **Projects:** Titanik Dataset. Adult Dataset
* **What I did:** Used Decision Trees method and predict whether a specific person would have survived the crasj or not. For the second dataset i used the same algorithm to predict whether a person's salary is greater or lower than 50k. 
* **Notebook:** [https://colab.research.google.com/drive/1U6tHiIkIYLbGZrmUn1UOLrLXEsrnd2UP]

### 🔹 Naive Bayes
* **Project:** MIlitary or Economy prediction.
* **What I did:** Implemented a probabilistic classifier to sort text data into categories. Learned the basics of text tokenization. Succesfuly created a model which predicts whether an article is about military or economy.
* **Notebook:** [https://colab.research.google.com/drive/1ASF0iCGkLqofHE9r-dFQ6balVWshede5#scrollTo=EfCJSrlsyWJC]

### 🔹 Ensemble Methods 
* **Project:** Adult Dataset
* **What I did:** Combined multiple machine learning models to create a stronger, more accurate predictor. Used an Encoder, split the data into 2 parts test and train, scaled the data and learned how bagging reduces errors.
* **Notebook:** [https://colab.research.google.com/drive/1t5E2GYAUkEwHkkNt_Oo3M_kEYFYj3c7i#scrollTo=fwnN622N-4JL]
