# ML-Projects

Welcome! This repository is a collection of my projects where I am learning and experimenting with different Machine Learning algorithms using Python and Google Colab.

## 🧠 Algorithms Explored

### 🔹 Linear Regression
* **Project:** Predicting Californian House Prices
* **What I did:** Used a dataset of housing features to predict prices. Learned how to fit a trend line and evaluate it using Mean Squared Error (MSE), mean absolute error (MAE) and R² score (R²).
* **Notebook:** [https://colab.research.google.com/drive/1pWC5tyU9RBrBzIafih644NXBf1h9GbPR]

### 🔹 Logistic Regression
* **Project:** Prediction of Loan Risk.
* **What I did:** Built a binary classifier to detect the risk of taking the loan.
* **Notebook:** [https://colab.research.google.com/drive/1pWC5tyU9RBrBzIafih644NXBf1h9GbPR]

### 🔹 Decision Trees
* **Projects:** Titanic Dataset. Adult Dataset
* **What I did:** Used Decision Trees method to predict whether a specific person would have survived the crash or not. For the second dataset, I used the same algorithm to predict whether a person would earn more or less than $50k based on demographic features.
* **Notebook:** [https://colab.research.google.com/drive/1U6tHiIkIYLbGZrmUn1UOLrLXEsrnd2UP]

### 🔹 Naive Bayes
* **Project:** Military or Economy prediction.
* **What I did:** Implemented a probabilistic classifier to sort text data into categories. Learned the basics of text tokenization. Successfully created a model which predicts whether an article is related to military or economy topics.
* **Notebook:** [https://colab.research.google.com/drive/1ASF0iCGkLqofHE9r-dFQ6balVWshede5#scrollTo=EfCJSrlsyWJC]

### 🔹 Ensemble Methods 
* **Project:** Adult Dataset
* **What I did:** Combined multiple machine learning models to create a stronger, more accurate predictor. Used an Encoder, split the data into training and test sets, scaled the data and learned how ensemble techniques improve model robustness.
* **Notebook:** [https://colab.research.google.com/drive/1t5E2GYAUkEwHkkNt_Oo3M_kEYFYj3c7i#scrollTo=fwnN622N-4JL]
